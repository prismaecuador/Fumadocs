# Guía de Inicio Rápido

Este es un ejemplo de cómo se ve una sección importada desde `/import/sections`.

## Requisitos

- Node.js 18+
- pnpm 8+

## Instalación

```bash
pnpm install
```

## Desarrollo

```bash
pnpm dev
```

Luego abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Estructura de Carpetas

- **src/app** - Páginas y rutas de la aplicación
- **src/content** - Contenido MDX generado automáticamente
- **src/components** - Componentes React reutilizables
- **import/sections** - Carpetas con documentación a importar
- **import/config.json** - Configuración de marca y colores

## Próximos Pasos

1. Agrega tus propias secciones en `import/sections`
2. Ejecuta `pnpm ingest` para generar el contenido
3. Personaliza `import/config.json` con tu marca
